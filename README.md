# free-for.dev

开发者和开源作者现在有许多服务提供免费的层级，但找到它们需要花费时间来做出明智的决定。

这是一个软件（[SaaS](https://zh.wikipedia.org/wiki/%E8%BD%AF%E4%BB%B6%E6%9C%8D%E5%8A%A1)（软件即服务），[PaaS](https://zh.wikipedia.org/wiki/%E5%B9%B3%E5%8F%B0%E6%9C%8D%E5%8A%A1)（平台即服务），[IaaS](https://zh.wikipedia.org/wiki/%E7%B5%AE%E5%85%B7%E6%9C%8D%E5%8A%A1)（基础设施即服务）等）和其他提供免费开发层级的服务的列表。

本列表的范围仅限于基础设施开发者（系统管理员，[DevOps](https://zh.wikipedia.org/wiki/DevOps) 实践者等）可能认为有用的东西。我们喜欢所有的免费服务，但最好保持主题。如果有灰色地带，这个列表将会变得主观化；如果我不接受您的贡献，请不要感到冒犯。

本列表是由1500多个人做的 [Pull Requests](https://github.com/ripienaar/free-for-dev)请求、审查、想法和工作结果。您也可以通过发送 [Pull Requests](https://github.com/ripienaar/free-for-dev)请求来添加更多服务或删除已更改或不再提供的服务来帮助我们。

[![Track Awesome List](https://www.trackawesomelist.com/badge.svg)](https://www.trackawesomelist.com/ripienaar/free-for-dev)

**注意**：本列表仅适用于作为服务的提供，而不适用于自托管软件。要有资格纳入本列表，服务必须提供免费的层级，而不仅仅是一个免费试用。免费层必须至少持续一年，如果它是基于时间分区的。我们还考虑免费层的安全性，因此 [SSO](https://zh.wikipedia.org/wiki/%E5%96%8C%E9%80%9A%E7%99%BB%E5%BD%95) 没问题，但我不会接受将 [TLS](https://zh.wikipedia.org/wiki/%E6%9C
