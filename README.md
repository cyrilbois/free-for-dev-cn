# free-for.dev

开发人员和开源作者现在有很多提供免费层次的服务，但是要找到所有这些服务需要花费时间才能做出明智的决定。

这是一份包含免费开发者层次的软件（SaaS、PaaS、IaaS等）和其他服务列表。

这份列表的范围仅限于那些对基础设施开发人员（系统管理员、DevOps实践者等）可能有用的内容。我们喜欢所有的免费服务，但最好还是保持主题相关。有时候这是一条灰色界线，所以这是有偏见的；如果我不接受您的贡献，请不要感到冒犯。

这份列表来自于来自超过1600人的Pull请求、评论、想法和工作成果。您也可以通过发送Pull请求来添加更多服务或删除那些服务提供已经更改或被淘汰的服务。您可以点击此处查看更多信息：[Pull Requests](https://github.com/ripienaar/free-for-dev)。

[![跟踪精彩列表](https://www.trackawesomelist.com/badge.svg)](https://www.trackawesomelist.com/ripienaar/free-for-dev) 

**注意**：这份列表仅限于作为服务提供的服务，不包括自托管软件。要符合条件，服务必须提供免费层次，而不仅仅是免费试用。如果免费层次有时间限制，则至少必须提供一年的免费服务。我们还会从安全性的角度考虑免费层次，所以单点登录是可以的，但我不接受仅对付费层次提供TLS支持的服务。
